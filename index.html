Reachy Mini Gemini Live App

Real-time voice and video conversations with Reachy Mini using Google's Gemini Live API

Features

  • Real-time voice conversation via Gemini Live API
  • Camera streaming - Reachy Mini sees and understands its environment
  • Robot audio - Use Reachy Mini's microphone and speaker
  • Expressive movements - Head movements and antenna animations
  • Emotion expressions - Happy, sad, surprised, curious, excited, sleepy
  • Wireless support - Works with wireless Reachy Mini

Requirements

  • Reachy Mini robot (or simulation)
  • Google API key with Gemini access
  • Set GOOGLE_API_KEY environment variable

Usage

Once installed via the Reachy Mini dashboard, the app runs automatically. Configure your Google API key in the environment before starting.

export GOOGLE_API_KEY=your-key-here

Audio Modes

Local Audio

Uses your computer's microphone and speakers

Robot Audio

Uses Reachy Mini's built-in mic and speaker via WebRTC

Powered by Google Gemini Live API and Reachy Mini SDK
